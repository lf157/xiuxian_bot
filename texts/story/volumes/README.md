## 核心NPC

- **沈无极**：至交好友，废灵根→鸿蒙道，贯穿全篇
- **韩苍**：宗门兄弟，铁拳硬汉，忠义化身
- **楚千殇**：剑道天才，从复仇到守护的蜕变
- **柳清竹**：道侣，太阴灵根，太清宗叛逃者
- **王逆**：逆天殿少主，对手变战友
- **星璇**：星辰之女，万年存在，命运指引者
- **太初**：亦敌亦友，三千年执念，最终赎罪
